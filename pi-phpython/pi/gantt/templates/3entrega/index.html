O que é um grafico de gantt?

O Gráfico de Gantt, também conhecido como Diagrama de Gantt, é uma ferramenta visual para controlar o cronograma de um projeto ou de uma programação de produção, ajudando a avaliar os prazos de entrega e os recursos críticos.

- Como criar uma pessoa? Ao ir até a NAVIBAR, clicar na aba esquerda, selecionar "Cadastro de pessoas", feito isso, será gerido um código para a pessoa e será necessário atribuir um nome á ela uma forma de conta, como por exemplo, número de celular ou email.

- Como criar um projeto? Ao clicar no primeiro icone, da esquerda para direita, acima do gráfico, será aberto uma tela para cadatro de projetos, onde, ao criar um projeto ele terá um código, e será necessária dar um nome, uma cor, e, informar a data inicial do projeto, assim como, seu prazo final.

- Como criar uma tarefa? Ao clicar no segundo icone, da esquerda para direita, acima do gráfico, será aberto uma tela para cadatro de tarefa, onde, ao criar uma tarefa ele terá um código, e será necessária dar um nome, informar a qual projeto essa tarefa pertence, se ela tem alguma dependência de outra tarefa, e também sua data inicial e o prazo de entrega.

- Como atribuir pessoas a uma tarefa? Ao clicar no terceiro icone, da esquerda para direita, acima do gráfico, será aberto uma janela, onde o usuário terá de selecionar um projeto, selecionar uma tarefa, e selecionar a(s) pessoa(s) responsável(is) pela tarefa.
